Timestamp Microservice API

This is an API that convert a given calendar date to it's timestamp.

Example Usage

Get the current date

To get the data for the current date:

[your_url]/api

Get other dates

To get the data for a specific date, a date format (YYYY-MM-DD) is needed, such as:

[your_url]/api/1992-01-13

Or a valid unix timestamp, such as:

[your_url]/api/695309700000

Example Output

{ unix: 695309700000, utc: "Mon, 13 Jan 1992 13:35:00 GMT" }
